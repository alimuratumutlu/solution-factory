# Contributing

Solution Factory is early. The best contributions are small, testable, and
focused on the local pipeline runner.

## Good First Areas

- Improve the React Flow canvas
- Add node configuration forms
- Expand the pipeline schema
- Implement shell node execution
- Improve example pipelines
- Add tests for schema validation

## Development

```bash
npm install
npm run dev
npm run build
```

Desktop development requires Rust and Tauri prerequisites:

```bash
npm run tauri dev
```
